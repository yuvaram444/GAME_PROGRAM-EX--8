# GAME_PROGRAM-EX--8

### Name : YUVARAM S
### Reg No : 212224230315

## Aim :

To create and edit basic level landscape, add foilage and landscape texture.

## Software Required :

1. Unreal Engine - v5.3 or v5.5

## Procedure :

1. Create a Basic Level on a Unreal Engine Project, Select Landscape mode and edit the landscape terrain.

2. Add foilage to the landscape using the Foilage Mode.

3. Add an existing mesh or Import your own mesh for the foilage.

4. Add textute for the Landscape.
